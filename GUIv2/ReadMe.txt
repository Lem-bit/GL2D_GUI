{
  Модули GUI можете использовать где хотите, и как хотите они абсолютно бесплатные. 
  Но в знак благодарности можно отправить некую сумму на карту сбера: 4274 3200 5194 9090 ))
  
  К сожалению нормально компоненты никто не тестировал, так что возможны некоторые
  баги в работе, если вы нашли проблему, то напишите мне на почту, постараюсь исправить.
  
  Самая актуальная версия исходников находится вот тут https://github.com/Lem-bit/GL2D_GUI 
  обновляется чаще чем на сайте lemgl.ru
  
  Проблемы которые возможно будут очень долго жить:
  - Компонент не может быть обрезан если заходит за границы формы, вы сами должны за этим следить
    (нужно отрисовывать в буфере, а затем вырезать)
  - Нет автоматического выравнивания (Align)
  - В загрузчике текстур не проверялась загрузка JPG, JPEG, TGA, PNG текстур из ресурсов
  - Везде где есть списки ListBox, ComboBox, Table... основаны на TList который приведет к Out of Memory если 
    добавить много объектов (около 10000), больше подходит для работы с маленьким списком
}
